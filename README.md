# RHNet
This is a lightweight network for crowd counting. 

## Environment
tensorflow 1.12.0; python 3.6 
Gpu: Gtx1070Ti

## Data 
Get to this [data](https://mailnwpueducn-my.sharepoint.com/:f:/g/personal/gjy3035_mail_nwpu_edu_cn/EkxvOVJBVuxPsu75YfYhv9UBKRFNP7WgLdxXFMSeHGhXjQ?e=IdyAzA)
Thanks to `{Gao, Junyu and Gao, Chenyu and Zeng, Xin and Qin, Zequn and Wen, Jun}` 's work in [C^3-Framework](https://github.com/gjy3035/C-3-Framework)
### Besides
I made a simple dataset of Melon seed、duck、horse and..[link](https://github.com/yurui777/Data-dense-objects)

## Train and Test
Just change the path of data and some network parameters in train.py, test.py ,then:
run `python train.py`
run `python test.py`
